A clean, minimal social media platform to connect with your community.

## Features
- **Modern Auth**: Secure Email & Password or Magic Link logins via Supabase.
- **Home Feed**: Real-time reverse chronological feed with Stories.
- **Posting**: High-quality image sharing with privacy controls (Public/Friends Only).
- **Social Connectivity**: Add friends and manage requests to see private posts.
- **Fast Search**: Instant username-based discovery.
- **Personal Profiles**: Custom bios, avatars, and post grids.

## Tech Stack
- **HTML5 & Vanilla CSS**: Modern, premium design system with CSS variables.
- **Vanilla JavaScript**: Lightweight SPA-style routing and logic.
- **Supabase**: Cloud database, Authentication, and Image Storage.

## How to Run
1. **Directly**: Open `index.html` in any modern web browser.
2. **Local Server (Recommended)**: Run `python -m http.server 8000`.
